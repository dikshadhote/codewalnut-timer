# Timer App Assignment

## **Project Setup**

1. Clone the repository:  
   ```bash
   git clone git@github.com:dikshadhote/codewalnut-timer.git
   ```

2. Install dependencies:  
   ```bash
   npm install
   ```

3. Start the development server:  
   ```bash
   npm run dev
   ```

4. Run tests:  
   ```bash
   npm vitest
   ```
